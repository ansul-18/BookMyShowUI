# 🎬 BookMyShow Frontend (UI)

## 📌 Project Overview

This is the frontend (UI) of the BookMyShow Clone project.
It allows users to browse movies, view shows, and book tickets online.

---

## 🚀 Features

* 🎥 View Movies
* 🏢 View Theatres
* ⏰ Show Timings
* 🎟️ Book Tickets
* 🔐 User Login / Signup

---

## ⚙️ Tech Stack

* HTML
* CSS
* JavaScript

* Payment Integration
* Better UI/UX
* Mobile Responsive Design
